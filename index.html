Flexbox The Flexbox, short for the CSS Flexible Box Module, is a one-dimensional layout model within CSS that makes it simple to layout your page, application, widget, or gallery, or just any resposive feature your site requires.
Grid The CSS Grid Layout Module, or CSS Grid, is a powerful grid-based layout system in Cascading Style Sheets that makes it simple to design complex resposive web layouts across browsers using a two-dimensional grid.

BootstrapDevelopers describe Bootstrap as "Simple and flexible HTML, CSS, and JS for popular UI components and interactions". Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
TailwindTailwind CSS is detailed as "A utility-first CSS framework for rapid UI development". Tailwind is different from frameworks like Bootstrap, Foundation, or Bulma in that it's not a UI kit. It doesn't have a default theme, and there are no build-in UI components.

CSS Box ModelEvery element that can be displayed on a web page is comprised of one or more rectangular boxes. CSS box model typically describes how these rectangular boxes are laid out on a web page. These boxes can have different properties and can interact with each other in different ways, but every box has a content area and optional surrounding padding, border, and margin areas.
